# Rooster's tail Game
# Overview
Lost Pages is a story-driven exploration game set in a dark, distorted fairy-tale world.
The player awakens in a mysterious place with no memory of how they got there. As they explore the environment, they encounter unusual creatures and uncover fragments of a broken story.
Scattered pages reveal a disturbing truth — someone is trying to rewrite or destroy the narrative itself.

# Features
- Page collection system (affects ending)
- NPC interactions (Penguin, Deer, Horse, Tiger)
- sound system (implementation in blueprints)
- Trigger-based interactions
- NPC behavior and simple AI
- Level scripting

## Gameplay
Explore a dark, atmospheric world
Interact with unique NPCs
Discover hidden story pages
Experience environmental storytelling
Survive a final chase sequence

# Tech Stack
Engine: Unreal Engine (UE)
Language: C++
Visual Scripting: Blueprints
